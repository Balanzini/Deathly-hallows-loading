# Deathly hallows indeterminate loading

Basic indeterminate loading for Android.


## Custom attributes
Set the color:
```java
app:color="@color/loading_color"
```

Set time to complete the cycle, in milliseconds:
```java
app:period="200"
```

Set the stroke width:
```java
app:stroke_width="7dp"
```
